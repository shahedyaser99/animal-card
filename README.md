## Animal Card

A simple HTML/CSS project that displays an animal card with the following features:

- Animal name aligned to the left
- Image of the animal
- Interesting fact in italic
- List of animal details (Habitat, Diet, Lifespan) with bold labels
- Borders around the card and information box
- Fixed card width with spacing around the image
- Clean layout and spacing between elements

## How to View

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Enjoy viewing the animal card!

## Files

- `index.html` - main HTML file
- `style.css` - CSS styles
- `Octopus.jpg` (or any image you want) - sample image of the animal

## Customization

- You can change the animal name, image, and details in `index.html`.
- The card style can be customized in `style.css` (colors, spacing, borders). animal-card
